## üîó **Backend Integration**

The mobile app communicates with the Guardix FastAPI backend. Default configuration:

- **Base URL**: `http://10.0.2.2:8000/` (Android emulator)
- **Default User ID**: `guardix-device`

### **Active API Endpoints**
- `POST /auth/login` ‚Äì JWT authentication for secure calls
- `POST /scan/apk` ‚Äì ML-powered malware detection for installed apps
- `POST /scan/phishing` ‚Äì NLP-based URL and text analysis
- `POST /auth/biometric` ‚Äì Behavioral biometric validation
- `POST /monitor/ids` ‚Äì Network telemetry for intrusion detection
- `GET /models/` ‚Äì Lightweight model metadata inspection

### **End-to-End Setup**
1. Start backend: `uvicorn app.main:app --reload --host 0.0.0.0 --port 8000`
2. Configure API_BASE_URL in `app/build.gradle.kts` if needed
3. Launch Android app for live ML-powered features
